Poke-Survey

A website to vote for your favorite Pokemon made with Express, MongoDB, and Mongoose.

Site Code

Voting
Mongoose and MongoDB store the votes.

Updating Votes
The back end creates a hash unique to each user's browser to show the user's previous votes and to let them update their votes.

View
EJS is used for the template and to loop through the data to show the images and voting results.

Styling
Bootstrap is used to style the website and display the results with small bar charts.
